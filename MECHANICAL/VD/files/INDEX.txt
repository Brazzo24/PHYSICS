💡 KEY CONCEPTS TO UNDERSTAND
=============================

1. SymPy Basics
   → Symbols (model.a_y, model.phi, etc.)
   → Expressions (combinations of symbols)
   → Equations (expressions = 0)
   → Find in: USAGE_GUIDE.md

2. Numerical Evaluation
   → .subs([...]) replaces symbols with values
   → .evalf() evaluates numerically
   → Example: INTERACTIVE_EXAMPLES.py Example 1

3. Roll Dynamics
   → Simple: φ = arctan(a_y/g)
   → Complex: Load transfer nonlinear
   → Key insight: Height/CoM position matters!
   → Find in: USAGE_GUIDE.md "Key Insights"

4. Tire Constraints
   → Forces limited by friction: √(F²+S²) ≤ μ·N
   → Affects max cornering speed and acceleration
   → Find in: ADVANCED_PATTERNS.py Pattern 3

5. Performance
   → Use lambdify() for >10 evaluations
   → Symbolic solving is exact but slow
   → Numerical solving is approximate but fast
   → Find in: ADVANCED_PATTERNS.py Pattern 1
