💡 KEY INSIGHTS
===============

1. Roll angle (Eq. 3) is SIMPLE: φ = arctan(a_y / g)
   → For a_y = 5 m/s² and g = 9.81 m/s² → φ = 27°

2. Load transfer (Eq. 4) is NONLINEAR
   → Not just about CoM position, also about roll inertia
   → sqrt(1 + a_y²) term shows coupling

3. Tire forces (Eq. 5-6) are CONSTRAINED by friction
   → Can't exceed μ·N in any direction
   → Front and rear work together (Pacejka friction ellipse)

4. For your chicane/Mozzi-axis work:
   → Use this to validate your roll angles
   → Use load transfer to check wheel contact
   → Use force constraints to check traction limits

📞 TROUBLESHOOTING
==================

"ImportError: No module named sympy"
→ Run: pip install sympy numpy

"SyntaxError or attribute error"
→ Make sure you're in the same directory as motorcycle_dynamics_solver.py
→ Or add it to your Python path

"Evaluation is slow"
→ You're using .subs() in a loop → Use lambdify() instead!
→ See ADVANCED_PATTERNS.py Pattern 1

"Need to solve for an unknown"
→ Use sp.solve(equation, unknown) for symbolic solution
→ Use scipy.optimize.fsolve() for numerical solution
→ See ADVANCED_PATTERNS.py Pattern 2
